# Sistema de listagem de compras
Este projeto implementa um CRUD em um sistema de criação de lista de compras (Criar e adicionar item, listar, atualizar e remover) através da metodologa TDD na linguaem Python.
# Introdução 
O TDD (Test-Driven Development) é um método de desenvolvimento onde você escreve o teste automatizado antes do código funcional. Ele funciona em um ciclo rápido de 3 passos:
- Red: Testes para uma nova função na espera de uma falha
- Green: Códigos mínimos e simples para passar em testes
- Refactor: Melhoria do código criado, eliminando sujeiras e duplicações.
O objetivo principal é guiar o design do software, gerando códigos mais limpos, organizados e livres de bugs desde o início.
# Funcionalidades do sistema
- Criar um novo item
- Listar os itens da lista de compras
- Atualizar as informações de um item
- Remover um item
# Tecnologias utilizadas
- Linguagem Python 3+;
- Biblioteca pytest;
